vue-cli的使用，教程，上手，vue脚手架的使用

发表于 2016-08-17 | 分类于 vue

注：vue-cli 是一个vue官方发布vueJS项目脚手架：点击查看，里面集成了webpack打包工具。

1.要求已安装Node.js (>=4.x, 6.x preferred) and Git.
可以设置cnpm可以提升依赖包下载速度：
（以后就可以完全使用cnpm）
这里默认你已安装好nodejs了啊

2.安装vue-cli

$ cnpm install -g vue-cli
注：-g为全局安装，如果是mac按如下写法：
sudo npm install -g vue-cli

3.创建模板项目
命令格式：vue init <template-name> <project-name>
其中template-name是可选模板项，project-name是创建项目的名称。
参考一下图片

4.然后

npm istall
npm run dev
就会自己在浏览器打开

生成的目录结构如下：

# vue-cli脚手架 # vue # vue-cli
使用vue.js实现checkbox的全选，和多个的删除
webpack打包路径问题，生成相对路径
